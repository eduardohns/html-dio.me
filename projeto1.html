Estrutura basica do html

A estrutura basica do html é o html lá no topo e dentro dessa estruta que é o html fica o head e o body.

O head que seria a cabeça do site onde fica o nome do site,que aparece la em cima no navegador e dentro do head fica também o meta que nele fica tipo: a compartibilidade do html com ispositiveis moveis,os caracteres que o html suporta

O body é o corpo do HTML, onde vai os parágrafos, as tags, os links para outras páginas, a parte visível do html é o body.

Falando sobre as tags

A tag i que deixa as palavras em italico

A tag strong que deixa as palavras em negrito

A tag input que da um espaço para que o usuario escreva ou coloque um numero ou clique com o mouse

A tag img que serve para colocar alguma imagem no html

Alguma tags não precisam de fechamento que é o caso da tag input e img

elementos das tags

Atributo id que significa a indentificação da tag,o que ajuda caso queira mudar a tag com o javascript ou edita com o css

O atributo style serve para colocar a cor na palavra, edita a fonte e tudo mais

Textos

O P que significa o parágrafo

O h1 ao h6 onde cada h1 é o titulo e h2 em diante são os sub-titulos e cada um respeitando a hierarquia do numero maior ser o titulo maior

O blockquote é usada para citações, cita uma frase de um pensador,cita o site de onde pegou a informação e etc.

o mark que deixa uma marcação onde voce escolher

Listas

Para fazer listas se usa a tag li, a dois tipos de listas as ordenadas e a não ordenadas.

Listas ordenadas

Lista ordenadas, a tag ol, significa que é uma lista que segue uma ordem numérica,ou seja 1,2,3,4 e por ai vai,da para colocar uma lista dento de outra

listas não ordenada

a lista não ordenada, a tag ul, é uma lista ondem não tem numero do lado,apenas um ponto
